Automated tester for the M3439 project.

It makes use of Trinamic TMCL motor & driver.

## Requirements
### Prerequisites - Code
- MPLAB X v5.45 or higher
- XC16 v1.61
- TEST-TRACK 4.2
### Prerequisites - PCB
- See BOM file
- pd-109-57
- PIC24FJ128GA306 @ 16MHz
- PCBA T3439
- Autodesk EAGLE
### Prerequisites - Frame
- Autodesk Fusion360
- See BOM file
- See [Setup](#Setup)
